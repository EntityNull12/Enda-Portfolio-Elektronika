eksperimen dioda

  1. Instalasi

    2. Untuk pemasangan komponen menggunakan Multisim dengan cara seperti berikut:

    1. Klik "Place Source" seperti di gambar bawah ini
    2. Di kotak dialog "Select a Componant" di bagian component, ketik "AC_POWER" lalu klik ok
    3. Untuk pemasangan resistor, klik "Place Basic" seperti di bawah ini
    4. Lalu muncul kotak dialog seperti di bawah ini, lalu pilih componant "1KOhm ... " lalu klik ok
    5. untuk pemasangan dioda nya, klik "Place Diode" seperti di bawah ini
    6. lalu pilih component dioda "1n5406" lalu klik ok
    7. buatlah rangkaian seperti di bawah ini. untuk penasangan groundnya sama seperti cara 1 dan 2
  2. Cara Melakukan Analisis Dioda
    1. Pilih tab Simuate, lalu pilih "Analyses and Simulation"
    2. Lalu muncul kotak dialog seperti dibawah ini dan ikuti pengaturan seperti di gambar
    3. Pilih tab Output, lalu masukan "V(1)" dan "v(2)" lalu klik save
    4. Setelah save, klik run
  3. Analisis
    1. Anoda

      2. Di anoda ini, titik minimumnya adalah -999.8956m (mili ampere) dan titik maksimumnya adalah 441.4851m (mili ampere).

    2. Katoda

      3. Di katoda ini, titik minimumnya adalah -999.8452m (mili ampere) dan titik maksimumnya adalah 968.3370m (mili ampere).
